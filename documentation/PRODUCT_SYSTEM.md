# Product System

Products support:

- Pricing.
- Features.
- FAQ.
- Roadmap.
- Screenshots metadata.
- Documentation.
- Releases.
- Downloads.
- Changelog entries.
- Licenses.
- Orders.

Product delivery uses:

- `ProductRelease`
- `ProductDownload`
- `DownloadEvent`

File metadata supports ZIP, JAR, PDF, EXE, and other asset types.
