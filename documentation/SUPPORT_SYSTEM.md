# Support System

Support tickets include:

- Status.
- Priority.
- Internal notes.
- Attachment metadata.
- Ticket timeline notes.
- Linked product.
- Linked license.
- Linked customer.

Supported statuses:

- Open.
- Waiting.
- In Progress.
- Resolved.
- Closed.

Future integrations:

- Discord ticket handoff.
- Email replies.
- Customer-visible timeline replies.
